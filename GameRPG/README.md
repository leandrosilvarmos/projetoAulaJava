 ## Sobre o jogo
 
 Esse projeto foi realizado, para um trabalho academico da instituição Centro Universitario Senac
o projeto consiste em um RPG básico, onde você poder realizar eescolhas de personagens e o propio sistema, 
ira realizar a batalha entre os personagens esolhidos.

 
 
 ## Criação do Jogo
 O jogo foi criado utilizando a linguagem Java,e foi utlizado as tecnicas já conhecidas, tais como 
  - Polimorfismo
  - Herença
  - Estrutura condicionais
  - Pacotes 
 	- utilizando essas técnicas, realizei a criação de classes para os personagens do jogo, 
 é o jogo sendo um RPG criei as classes de Armas é Armaduras, para que o usúario possas equipar 
 em seus personagens escolhidos. 
 
 ## Personagens , Armas é Armaduras
  Com as classes criadas, deixamos algumas opções criadas para a escolha do usuarios.
  os personagens foram dividos em 5 Categorias concretas sendo elas:
  - Humano
  - Dragões
  - Orcs
  - Seres Magicos
  - Guerreiro
  - dentro de cada categoria existe SubCategorias, que podem ser escolhidas
  Para as armas Criamos 5 Opções é seguimos o mesmo padrão para Armaduras

 ## Funcionamento 
 
 Para executar o projeto em sua maquina e simples, temos algumas regras nesse projeto que são elas
  -  Escolher 2 Armadura 
  -	 Escolher 2 Armas
  -	 Escolher 2 combatentes
  - assim voce pode começar uma batalha.
